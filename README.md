# 水牛云盘前端页面

## 一、架构
Vue2 + Electron +Element-UI


## 二、环境准备

### 1、下载安装node.js
下载地址https://nodejs.org/en/download/

### 2、代理安装
1.安装时临时指定代理：
npm --registry=https://registry.npm.taobao.org install

2.全局配置：
npm config set registry https://registry.npm.taobao.org


## 三、运行

```shell
#执行包安装
npm install
#安装结束运行程序
npm run dev
```
